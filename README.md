# **Subject Guide & Question Bank Assistant AI Agent**

**An AI-powered Academic Learning Assistant that helps students learn from lecture notes, textbooks, lab manuals, and previous-year question papers using Retrieval-Augmented Generation (RAG).**

## Overview

**Subject Guide & Question Bank Assistant AI Agent **enables students to upload multiple academic resources and ask questions in natural language. The system retrieves relevant information from uploaded documents and generates comprehensive answers with source references.

### Example Questions

* Explain Database Normalization with examples.
* How to solve Question 3 from the 2023 DBMS paper?
* Summarize Unit 4 for exam revision.
* Generate practice questions on SQL Joins.

---

##  Features

*  Multi-document upload (PDF, DOCX, PPTX)
*  Topic-based academic explanations
*  Semantic search using vector embeddings
*  Previous year question solving
*  Automatic content organization
*  Source citations for every response
*  Interactive Streamlit dashboard
